Grid

make a grid layout.

Method: float-minus-margin

<div class="grid_float-minus-margin">
  <li><div class="min-box green">
    1 <img src="http://lorempixel.com/800/800/" alt="" width="800" height="800">
  </div></li>
  <li><div class="min-box green">2</div></li>
  <li><div class="min-box green">3</div></li>
</div>
.grid_float-minus-margin {
  margin-right: -10px; // clear the extra gutter
  &:after { content: ''; display: table; clear: both; } // clear float
  li {
    float: left;
    padding-right: 10px; // gutter
    box-sizing: border-box;
  }
  > :nth-child(1) { width: 25%; }
  > :nth-child(2) { width: 35%; }
  > :nth-child(3) { width: 40%; }
}

    • White Space Below Image

    There is white space below image, iframe, object and embed which break the layout.

    <div class="white-space-below-image">
  <img src="http://lorempixel.com/100/100/" alt="rock-texture" width="100" height="100">
</div>
    .white-space-below-image {
  background-color: #f92672;
}

    Method: vertical-align

    IE6

    <div class="white-space-below-image_vertical-align">
  <img src="http://lorempixel.com/100/100/" alt="rock-texture" width="100" height="100">
</div>
    .white-space-below-image_vertical-align {
  background-color: #f92672;
  img { vertical-align: bottom; } // or top
}

    Method: display

    IE7

    <div class="white-space-below-image_display">
  <img src="http://lorempixel.com/100/100/" alt="rock-texture" width="100" height="100">
</div>
    .white-space-below-image_display {
  background-color: #f92672;
  img { display: block; }
}
